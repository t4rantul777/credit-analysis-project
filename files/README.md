# Анализ бизнес-процесса: Обработка кредитных заявок

Учебный проект по системному анализу. Включает моделирование бизнес-процесса, функциональные требования, REST API спецификацию и SQL-запросы для анализа данных.

## Структура

```
credit_analysis/
├── queries.sql          # SQL-запросы для анализа данных
├── requirements.md      # Функциональные требования и API спецификация
├── bpmn_description.md  # Описание BPMN-процесса
└── README.md
```

## Стек

- PostgreSQL — база данных
- Draw.io — моделирование BPMN-процесса
- REST API — JSON-формат обмена данными

## Ключевые запросы

- Распределение заявок по статусам
- Средняя сумма одобренных заявок
- Клиенты с рейтингом выше среднего
- Время обработки заявок
- Топ причин отказа
- Заявки без решения более 3 дней
